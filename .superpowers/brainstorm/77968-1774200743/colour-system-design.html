Colour Settings System - Design

Colour swatches appear inline within the Chart Options dialog, only for active elements

Approach

Add a ChartColours config object passed alongside existing options. Colour swatches render as small coloured squares with labels, grouped per active feature. Clicking a swatch opens the native <input type="color"> picker. Session-only (no persistence).

Conditional Visibility Rules

Candlestick colours → shown when chartType = 'candlestick'
Line colour → shown when chartType = 'line'
Area colours → shown when showArea = true
Volume colours → always shown (volume is always on)
SMA colour → shown when smaConfig.enabled = true
EMA colour → shown when emaConfig.enabled = true
